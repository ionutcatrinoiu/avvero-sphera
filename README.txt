SPHERA Firebase

1. Urcă întregul conținut al folderului pe Netlify, Vercel sau Firebase Hosting.
2. Autentificare administrator:
   utilizator: administrator
   parola: parola contului Firebase io.catrinoiu@gmail.com
3. La primul login, aplicația migrează automat datele locale existente în Firestore dacă baza este goală.
4. Date sincronizate live: comenzi, checkpointuri, clienți adăugați, nomenclatoare și profiluri utilizatori.
5. Fișierul firestore.rules conține regulile deja publicate.

Important: aplicația trebuie rulată prin HTTPS sau localhost, nu direct prin file://.
